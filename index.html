etaus

Random Number Generator

LICENSE

etaus is a random number generator (RNG) that uses 1024 32-bit states, and performs a Bays-Durham shuffle on those states.

Created by aquila62 at http://www.github.com

To learn more about this generator, go to the etaus project page.
