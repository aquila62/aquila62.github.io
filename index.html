eegl

Random Number Generator

LICENSE

eegl is a linear feedback shift register (LFSR) random number generator (RNG) that uses 1024, 32-bit states, and performs a Bays-Durham shuffle on those states.

Created by aquila62 at http://www.github.com

To learn more about this generator, go to the eegl project page.
